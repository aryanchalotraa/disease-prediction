# Airborne and Waterborne Disease Risk Prediction

## Description
This project predicts the risk of airborne and waterborne diseases using
real-time weather data of Indian cities.

## Input Parameters
- Temperature
- Humidity
- Rainfall

## Output
- High Airborne Disease Risk
- High Waterborne Disease Risk
- Low Disease Risk

## Technologies Used
- Python
- OpenWeatherMap API

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Add your API key in weather_api.py
3. Run: python main.py
